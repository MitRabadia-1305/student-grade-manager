# Student Grade Manager

A full-stack web app for schools to manage and track student grades.

## Features
- Add and delete students
- Auto-saves data across page reloads
- Real-time search by student name
- Auto-calculated statistics (total students, highest score, class average)
- Export data to JSON file
- Letter grade assignment (A, B, C, D, F)

## Built With
- HTML, CSS, JavaScript (Frontend)
- Python Flask (Backend)

## How to Run
1. Install dependencies:
    pip install flask-cors
2. Run the Flask server
3. Open `project.html` with Live Server in VS Code
